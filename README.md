# Audio2Text

## Project Status

⚠️ **This repository contains old/legacy code** ⚠️

All the code in this repository represents previous versions and experimental implementations. This codebase is no longer actively maintained or recommended for use.

### Repository Structure

The repository contains various audio-to-text conversion implementations and experiments:

- **Root files**: Various Python scripts for audio transcription using different approaches
- **st----final/**: Final versions and iterations of streaming implementations
- **streaming/**: Real-time audio processing and emotion detection experiments

### Note

This code is kept for historical reference and learning purposes. For current implementations, please refer to newer repositories or contact the maintainer.
